## 安装依赖:
    git clone https://github.com/FanHehe/backend.git
    cd backend
    cnpm install
## 运行
    npm run dev

## 需要

    安装 mysql：brew install mysql
    命令行执行 mysqld （保持服务运行，不退出）
    mysql 用户名 root 密码 root 或者自改 src/config.js : DB部分
    mysql 中 存在表 tbl_user

[点击打开](http://localhost:8081)
